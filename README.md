## Getting Started

First, run the development server:

Instalar Gulp de manera global: npm install --global gulp-cli

```bash
npm run dev
gulp dev //para que corra gulp para los estilos
# or
yarn dev
``` 

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
